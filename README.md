# Alien Report


## 소개
### CCTV를 통해 집안에서 발생하는 이상 현상들을 감시하고 보고해라 

## 개발 기간
- 24.02.26 ~ 24.03.04

## 멤버
- 이상민(팀장) : 프로젝트 관리, 카메라 배치, 이상현상 구현
- 김유원 : 사운드 작업 및 파티클 시스템 추가, 이상현상 구현
- 유호진 : 포스트프로세싱, 카메라 필터 및 광원효과 작업, 이상현상 구현
- 신채윤 : 게임 로직 구현, UI 작업

## 개발 환경
- Engine : Unity 2022.3.2f1
- Language : C#
